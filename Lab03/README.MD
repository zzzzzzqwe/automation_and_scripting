# Сборка и запуск контейнера с cron

Для того чтобы запустить контейнер с cron необходимо клонировать репозиторий и перейти в папку с проектом:

```bash
cd Lab03
```

И собрать и запустить контейнер
```bash
docker compose up -d --build
```
